## 💻 Projeto

Esse é um projeto web responsivel de uma cápsula do tempo para exibir memórias em uma linha de tempo.

## 🚀 Tecnologias

Esse projeto foi desenvolvido durante o "Next Level Week" da rocketseat com as seguintes tecnologias:

- HTML
- CSS
- Git e Github
